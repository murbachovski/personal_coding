## 🚀 프로젝트 설명
```
LMStudio 기반의 로컬 LLM 챗봇
<번역 기능 추가>
입력 : 한국어 => 영어
답변 : 영어 => 한국어
```

## 🐍 가상환경 생성
```
conda create -n lm python=3.10
pip install lmstudio
pip install pyfiglet
pip install termcolor
pip install deep-translator
```

## 💡 FastAPI 서버 실행
```
./run.sh
```
