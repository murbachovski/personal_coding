# 🚀 프로젝트 설명
```
Stable Diffusion 기반 Hugging Face API로 텍스트 입력 → 이미지 생성 후 iTerm2에 출력
```

# 🐍 가상환경 생성
```
conda create -n hugg python=3.10
pip install huggingface-hub pillow
```

# ▶️ 실행
```
python3 app.py
```
