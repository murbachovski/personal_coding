# 🚀 프로젝트 설명
```
Ultralytics YOLO11n-cls 기반으로 흉부 CT에서 5가지 질환을 분류하는 의료영상 특화 모델
```

# 🐍 가상환경 생성
```
conda create -n chestct python=3.10
pip install ultralytics
```
