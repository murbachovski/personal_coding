# 프로젝트 설명
```
업로드한 이미지를 자동으로 설명해주는 FastAPI 기반 이미지 캡셔닝 웹 애플리케이션.
```

# 가상환경 생성
```
# conda create -n likeimgtext python=3.10
# pip install fastapi uvicorn python-multipart transformers torch pillow
```

# 웹 실행
```
# uvicorn main:app --reload
```
